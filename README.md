# Today I Learned (React)
React.js, CSS and Supabase project.

## Scripts

* To start a local development server:
```
npm run dev
```
or
```
npm run start
```

* To prepare a production build:
```
npm run build
```

* To ZIP the production build and be ready to deploy:
```
npm run zip:build
```
